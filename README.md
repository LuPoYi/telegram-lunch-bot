# Telegram Lunch Bot

1.  與 [botfather](https://core.telegram.org/bots#botfather) 聊天取得telegram bot token
2.  cp config.yml.backup config.yml，並將token放入，及使用redis
3.  ruby main.rb

* 基本CURD可work

### 使用指令列表
* /help
* /list
* /add [name]
* /update [old_name] [new_name]
* /remove [name]
* /random    隨機取得
* /choose    隨機三選一
* /close     關閉keyboard
